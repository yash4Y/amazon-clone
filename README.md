# Amazon Clone

A responsive Amazon-inspired e-commerce website built using **HTML, CSS, and JavaScript**. This project replicates the basic UI of Amazon and demonstrates front-end development skills.

## Features

- Responsive design
- Amazon-style homepage
- Product listings
- Navigation bar with search
- Clean and modern UI

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Getting Started

```bash
git clone https://github.com/yash4Y/amazon-clone.git
cd amazon-clone
```

Open `index.html` in your browser or run the project using the VS Code Live Server extension.
